Advise the customer to change the XtraView Menu Transmission port settings to RF IN for the HD PVR decoder Follow these steps:
  • Press Menu
  • Go to Advanced Options
  • Go to Remote Control Setup
  • Go to Transmit Relay Port (Highlight and press OK)
  • Change to RF IN (Use the Left or Right arrows)
  • Press OK to Save
  • Exit all the way out
  • Verify customer primary and secondary smartcard numbers on primary environment
    • These checks will verify the smartcard information on Clarity and the viewing environment
    • I.e. F must match H and A must match G
    • Confirm if the Primary smartcard number showing on TV1 matches the Primary smartcard on Clarity.
    • Confirm if the Secondary smartcard number showing on TV 1 matches the Secondary smartcard number on Clarity.
If there is a conflict between decoder settings and Clarity In order to resolve this issue, do the following:
  • Unlink XtraView on Clarity
  • Ensure that XtraView Status for both decoders is "Ready but Inactive" or "Inactive" (if not inactive, send command to clear history)
  • Relink XtraView on Clarity
  • Check XtraView status on HD PVR on primary environment
Instructions for checking the XtraView status on an HD PVR decoder using the remote
  • Press Menu on remote
  • Select Advanced Options
  • Select XtraView Menu
  • If the status is on ready but inactive then unlink XtraView and relink on Clarity.
  • If the status is on active then go to the secondary decoder and ask customer to check the back of the decoder to confirm which RF Port the Heartbeat Cable is connected to on the secondary decoder. If RF Port is on RF IN then confirm if the decoder XtraView Menu Settings show RF In port also by following the below steps:
  • Go to Menu
  • Go to Advanced Options
  • Go to Remote Control Setup
  • Check the Remote Relay Port
If the settings show as RF OUT then change it to RF IN by following the steps below:
  • Go to Menu
  • Go to Advanced Options
  • Go to Remote Control Setup
  • Go to Remote Relay Port
  • Change the Setting to RF In using the Left or Right Remote Arrow Button
  • Press OK twice to Save
  • Exit the Screen
