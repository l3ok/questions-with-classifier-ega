When a customer is activating anew device and a new package a new agreement is created on clarity by going to add devices and packages select the requested package and the pro-rated amount will be given to customer to make payment for the new agreement.

  • Go to manage viewing
  • Go to add device and packages
  • Under smartcard or device number
  • insert or populate the device number
  • Confirm if the device and smartcard matches
  • Click ok on the pop up to add the device
  • Select the preferred package
  • Select the submit option
  • Click on activate
