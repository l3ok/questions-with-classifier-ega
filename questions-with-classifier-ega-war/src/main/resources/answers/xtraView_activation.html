For an existing customer:

  • Verify the customer on Clarity
  • Go to "Manage Viewing"
  • Ask the customer to confirm the primary device
  • Select link devices
  • Select device 1 as primary device and select device 2 as secondary device
  • Click on "ADD"
  • Go to activate packages
  • Select the primary package
  • Click submit button
  • Activate Xtraview

For new customers:

  • Add devices (then refer to step 1 from the above)
