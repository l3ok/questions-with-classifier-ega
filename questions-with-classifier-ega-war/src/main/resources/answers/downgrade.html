Downgrade is only scheduled for the end of customer's billing cycle which is the day before their payment date or on the day of payment it is done when a customer request to be put on a lower package due to affordability or various reasons and when they had requested to be upgraded for only a particular period.

  • Manage viewing
  • Select downgrade/upgrade
  • Select the smartcard number that you would like to downgrade
  • Confirm package
  • Select package
  • Generate the quote
  • Advice customer on downgrade date (billing date)
  • Click on the complete package
