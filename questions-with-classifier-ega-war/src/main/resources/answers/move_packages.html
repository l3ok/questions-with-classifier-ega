Packages are moved when you are loading a new decoder or if you have more than 1 decoder on the account. This allows you to move the current services the customer has without reloading a new service. It is important to note if the old packages does not have HDPVR services you will need to load the services on a compatible decoder.

  • Go to clarity
  • Go to manage viewing
  • Select move packages
  • On the drop list, select the active package to move (from device) to the inactive (to device on the next tab) smartcard.
  • click "move Package"
